## 1. このアプリでできること

- **8 つの主要市場銘柄をリアルタイム監視** (日経225先物 / ダウ・ナスダック・S&P 先物 / ドル円 / 原油 / VIX / 米10年債)
- **急変動を自動検知**: 設定した閾値を超えた値動きでビープ音 + 画面ハイライト + バナー通知
- **AI が「なぜ動いたか」を日本語で説明**: 4 時間分のニュースを参照して LLM が要因を要約
- **マクロニュースを 21 ソースから集約** (日本 7 + 英語 14)
- **AI チャット**: 「今の地合いは?」「VIX 上昇の影響は?」など自由質問
- **TradingView チャート埋め込み** (日経 225 用)
- **値がさ株 5 銘柄** (ファーストリテイリング、東京エレクトロン等) も裏で監視、急変時はアラート通知
- **新版が出たら自動更新通知**

## 2. インストール

### ダウンロード

https://github.com/sora-moyou/jp225-monitor/releases/latest

ページ下の **「Assets」** から `JP225.Monitor_X.X.X_x64-setup.exe` をダウンロード。

### インストール手順

1. ダウンロードした `.exe` ファイルをダブルクリック
2. **「Windows によって PC が保護されました」** と出たら:
   - 「**詳細情報**」をクリック
   - 表示された「**実行**」ボタンをクリック
   - これは無料配布アプリで一般的な警告で、害はありません
3. インストールウィザードに従う (既定のままで OK)
4. スタートメニュー → **JP225 Monitor** で起動

### 動作環境

- **Windows 10 / 11** (64-bit)
- **WebView2 Runtime**: Win 11 は標準で含まれます。Win 10 でも最近の Windows Update で入っているはずですが、万が一見つからない場合は [Microsoft Edge WebView2](https://developer.microsoft.com/microsoft-edge/webview2/) からダウンロードしてください

## 3. 初回セットアップ — API キーの設定

起動時に **設定モーダルが自動で開きます**。ここで LLM (AI) の API キーを 1 つ以上設定します。

> ⚠ **API キー未設定でもアプリは動きますが、「なぜ動いたか」の AI 説明と AI チャットが使えません。**
> ニュース翻訳機能も使えません (価格監視・アラート音・チャートは動きます)。

### おすすめ: Gemini (無料・高品質)

**無料で日次 1500 回まで使えます。** これだけで日常運用に十分。

1. https://aistudio.google.com/apikey にアクセス
2. Google アカウントでログイン
3. **「Create API key」** をクリック
4. 表示された `AIza...` で始まる文字列を **コピー**
5. JP225 Monitor の設定モーダル「Gemini」欄に貼り付け
6. **「保存」** クリック → モーダル右上の状態が 🟢 になれば成功

### オプション: Groq (無料・超高速、日次 14400 回)

激しく使う場合のフォールバック用。

1. https://console.groq.com/keys にアクセス
2. アカウント作成 → 「Create API Key」
3. `gsk_...` で始まる文字列をコピー
4. JP225 Monitor の「Groq」欄に貼り付け → 保存

### オプション: OpenAI (有料、$5〜)

最後の砦。プロバイダ自動切替が働くので、Gemini と Groq の両方がレートリミットに達した時にだけ使われます。

### 複数キーを設定するメリット

優先順位は **Gemini → Groq → OpenAI**。429 (レート制限) でブロックされたら次のプロバイダに自動切替されます。1 つだけだとブロック時にしばらく使えませんが、2 つ以上あれば実質常時利用可能。

## 5. 主な機能

### 5.1 価格監視

8 つの主要銘柄を **2 秒ごとに更新** (デフォルト)。価格カードに最新値と騰落率が表示されます。

- **背景色のハイライト** = 直近で大きく動いた銘柄
- **`---` 表示** = データ取得失敗 (Yahoo + Investing.com 両方ダウン時、復帰待ち)

### 5.2 急変動アラート

以下のいずれかで発火:

- **磁度** (magnitude): 5 分間で閾値を超える変動 → `[トレンド]` ラベル
- **傾き** (slope): 30 秒間で急峻な動き → `[フラッシュ]` ラベル

発火時の挙動:
- 価格カードが点滅
- ビープ音 (上昇=高音、下落=低音、§8 で要有効化)
- アラート欄にバナー追加
- LLM がニュースを参照して 1〜2 文で説明
- 同じ銘柄は **60 秒のクールダウン** (短時間で連発しない)

### 5.3 AI による「なぜ動いた」説明

アラート発火 30 秒後にバナー内に表示。例:

> ⚡ 日経225先物 ▲ +0.45% / 4分 [トレンド]
> 18分前のFOMC議事要旨ハト派寄りで米株先物↑につられ日経も上昇。

**🔄 ボタン** でクールダウンを無視して即再生成可能。

### 5.4 AI チャットボード

左下のチャット欄から自由質問できます。LLM には **現在の全銘柄価格** と **直近 15 件のニュース** が自動でコンテキストとして渡されます。

例:
- 「今の地合いを 3 行で」
- 「VIX が上がってる原因は?」
- 「ドル円急騰してるけど介入の可能性?」

### 5.5 ニュース翻訳

英語ニュース行の **「訳」** ボタンをクリックで日本語化。詳細は §9。

### 5.6 値がさ株のバックグラウンド監視

ファーストリテイリング、東京エレクトロン、アドバンテスト、ソフトバンク G、ダイキンの 5 銘柄も裏で監視中。価格カードには表示されませんが、これらが急変したら同様にアラートが出ます (日経への影響大のため)。

## 6. API ステータスドット (受信中の右にある 4 つの○)

5 秒ごとに自動更新される **データソースの稼働状況** インジケータです。

| ラベル | 意味 |
|---|---|
| **Y** | Yahoo Finance (価格データ取得元) |
| **G** | Gemini (LLM 第一候補) |
| **Gr** | Groq (LLM 第二候補) |
| **O** | OpenAI (LLM 第三候補) |

### 色の意味

| 色 | 状態 |
|---|---|
| 🟢 | 正常稼働中 |
| 🟡 | 一時停止中 (429 等のレートリミット、復帰待ち) |
| ⚪ | 未設定 (LLM のみ — ⚙️ から API キー未入力) |

### 各ドットに hover (マウス重ねる) で詳細

- 🟢: 「Yahoo: 利用可」「gemini: 利用可」など
- 🟡: 「Yahoo: fallback中 (残 2:35 / 13:47:50 復帰予定)」 ← Investing.com にフォールバック中
- 🟡: 「gemini: 待機中 (残 1:20 / 13:46:00 復帰予定)」 ← 429 でクールダウン
- ⚪: 「groq: 未設定」

### よくある状況

- **Y が 🟡 のまま**: Yahoo に拒否られた (短時間に多数アクセスで 429)。値段は Investing.com スクレイピングが代替しているので **画面の価格は引き続き更新されます**。5 分で自動再試行。
- **3 つの LLM が全部 🟡**: 全プロバイダがレート制限中。アラートが発火しても説明が出ない状態。少し時間を置くか、設定モーダルからプロバイダを追加。
- **G/Gr/O が全部 ⚪**: API キー未設定 = LLM 機能 OFF。価格監視・アラート音・チャートは動きますが「なぜ動いた」AI チャット・ニュース翻訳が使えない。

## 7. 設定モーダル (⚙️)

右上 ⚙️ クリックで開きます。

### API キー欄 (Gemini / Groq / OpenAI)

- **「設定済み」表示**: 既にキーが入っている。新しいキーを入力して「保存」で **上書き**
- **空欄のまま「保存」**: 既存キーは **変更されません**
- **キー削除**: 一度モーダル外でキーを再生成 → 古いものを別途無効化 (詳細は各プロバイダのコンソールから)

### 定期 API ポーリング設定

- **Price polling (ms)**: 価格取得間隔。デフォルト **2000**。Yahoo に 429 が多発する場合は **5000** 以上推奨
- **News polling (ms)**: ニュース取得間隔。デフォルト **60000** (= 60 秒)
- **Sidecar port**: 内部 HTTP サーバのポート。**変更しない方が無難** (アプリ再起動が必要 + 配布版で完全反映は今後の課題)

範囲外の値を入れると「out of range」エラーになります。

## 8. アラート音を有効化する

**重要**: ブラウザ/Webview のセキュリティ仕様で、**音はユーザー操作後でないと鳴りません**。

初回起動時、画面右下に **「🔔 サウンドを有効化」** ボタンが出ます。1 回クリックすれば消えて、以降の急変アラートで自動的にビープが鳴ります。

(クリックしないとアラートが視覚的にしか出ない = 気づきにくい)

## 9. ニュース翻訳機能

英語ニュース行の **「訳」** ボタンクリックで日本語化:

```
変更前: [EN] CNBC 14:32 [訳]
        Powell signals possible rate cut at next FOMC meeting

変更後: [EN] CNBC 14:32
        パウエル氏、次回 FOMC で利下げの可能性を示唆
```

### 動作仕様

- **日本語ニュース** ([JP]) には [訳] ボタンは出ません
- 同じタイトルは 2 回目以降 **キャッシュから即返却** (LLM 消費なし、新着ニュース更新で再描画されても再翻訳しない)
- 翻訳の元になっているのは **タイトル文字列のみ** (本文は元の URL リンクのまま、クリックすると原文ページが別タブで開く)
- LLM API キー未設定だと動きません

### コスト

1 タイトル ≈ 300 token 程度。Gemini 無料枠 (1500 req/日) で 1 日 1500 回以上は翻訳できる計算。普通に使う分には全く減らないでしょう。

## 10. 自動アップデート

新バージョン (v0.3.1 など) が出ると、**アプリ起動 5 秒後** に右上にトーストが表示されます:

```
┌──────────────────────────┐
│ 🆙 v0.3.1 が利用可能です │
│ - 改善点の説明...         │
│ [更新] [後で]            │
└──────────────────────────┘
```

- **「更新」**: ダウンロード (進捗バー表示) → 自動再起動 → 新版で起動 (約 30 秒)
- **「後で」**: 24 時間後に再通知。それまでトーストは出ません

## 11. トラブルシューティング

### Q. 「切断」表示が消えない / 価格が出ない

1. アプリを完全終了 (タスクトレイにアイコンがあれば右クリック → 終了)
2. 再度スタートメニューから起動
3. それでも直らない場合は **再インストール** (上書きで OK、設定は保持)

### Q. アラート音が鳴らない

下部の **🔔 サウンドを有効化** ボタンを 1 回クリックしましたか?

### Q. アラート発火しても「なぜ動いた」説明が出ない

1. ⚙️ で API status を確認
2. **全プロバイダ ⚪** = API キー未設定 → §3 を参照して設定
3. **全プロバイダ 🟡** = 全レートリミット中 → 数分待つ。頻繁に起きるなら別プロバイダを追加

### Q. **Y** ドット (Yahoo) が黄色のまま

Yahoo Finance に拒否されています (cookie / レートリミットの揺らぎ)。価格データは Investing.com フォールバックが代替中で、画面の価格は引き続き更新されます。5 分で自動的に Yahoo に再試行します。

### Q. 価格カードが `---` のまま

両ソース (Yahoo + Investing.com) が同時失敗中。ネット接続を確認、数分待つ、ダメなら再起動。

### Q. 起動時にファイアウォール許可ダイアログが出る

「アクセスを許可する」で OK。内部 HTTP サーバ (port 3000) のために必要です。

### Q. ログを見たい

📋 ボタン → サーバログモーダル。直近 200 行を表示、新しいログが下に自動追加されます。

### Q. インストール時に SmartScreen 警告

無料アプリ全般の警告です。「詳細情報」→「実行」で OK。気になる場合は配布元 (sora-moyou/jp225-monitor) のリポジトリで内容を確認できます。

## 12. データの保存場所

| 項目 | 場所 |
|---|---|
| 設定 (API キー、polling 間隔等) | `%USERPROFILE%\.jp225-monitor\config.json` |
| ログ | アプリ内 📋 モーダルのみ (アプリ再起動で消える、ファイル保存しない) |
| アプリ本体 | `C:\Program Files\JP225 Monitor\` (またはインストール時に指定した場所) |

`config.json` は **テキストファイル** なのでメモ帳で開けます (但し編集して構造を壊すと起動失敗する可能性あり、慎重に)。
