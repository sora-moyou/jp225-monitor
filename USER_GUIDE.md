## クイックスタート (3 ステップ)

### 1. ダウンロード

https://github.com/sora-moyou/jp225-monitor/releases/latest

ページ下の **Assets** から `JP225.Monitor_X.X.X_x64-setup.exe` をダウンロード。

### 2. インストール

- ダブルクリック → **「Windows によって PC が保護されました」** で「**詳細情報**」→「**実行**」
- インストールウィザードは既定のままで OK

### 3. API キー設定

起動時に設定モーダルが自動で開きます。

- [Gemini 取得 (無料、推奨)](https://aistudio.google.com/apikey) → `AIza...` をコピー → モーダル「Gemini」欄に貼り付け → 保存
- 詳細: Web 版マニュアル §3

## 主な機能

- **8 銘柄リアルタイム監視** (NIY=F / NQ=F / YM=F / ES=F / JPY=X / CL=F / ^VIX / ^TNX) を 2 秒間隔
- **適応 z-score 急変検知** (v0.3.17 / v0.3.19) — 1 分窓 burst + 5 分窓 trend + 日経 225 専用 5〜10 秒 tick の 3 層。過去 60 分 σ 基準で相場局面に追従。**v0.3.19 以降アラートを出すのは日経225先物 (NIY=F) のみ** — 他銘柄は横断確認・AI 材料に
- **AI 自動説明** — Gemini → Groq → OpenAI 自動フォールバック。急変時は大きく動いた他資産 (他指数・FX・商品・値がさ株のうち |z|≥4.0) も判断材料に
- **AI チャット** — 全銘柄価格 + 直近 15 件ニュースをコンテキストに自由質問
- **21 ニュースソース** — 日本 7 + 英語 14、英語見出しはワンクリック翻訳
- **TradingView チャート埋め込み** (日経 225)
- **音声アラート** — 上昇=高音、下落=低音、初回 1 回手動有効化
- **自動アップデート** — 起動 5 秒後にチェック、トースト通知。設定画面の「最新かチェック」で手動確認も可能 (v0.3.18〜)

## トップバーの 4 つの○ (API ステータスドット)

5 秒ごとに自動更新されるデータソース稼働状況:

| ラベル | 意味 |
|---|---|
| **Y** | Yahoo Finance |
| **G** | Gemini |
| **Gr** | Groq |
| **O** | OpenAI |

色: 🟢 正常 / 🟡 待機中 (429) / ⚪ 未設定。hover でカウントダウン表示。

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| アラート音が鳴らない | 画面右下「🔔 サウンドを有効化」を 1 回クリック |
| 「切断」のまま | アプリ完全終了 → 再起動 |
| AI 説明が出ない | ⚙️ で API キー状態確認、未設定なら設定 |
| Y ドットが 🟡 | Investing.com フォールバック中、5 分で再試行 |
| 価格カードが `---` | 両ソース失敗、ネット確認 / 再起動 |
| インストール時 "Error opening file" | v0.3.3 以降は自動解消。旧版なら タスクマネージャ で `jp225-sidecar.exe` を終了して再実行 |
| アラートが頻発する | v0.3.7 以前は内部の JPY 換算バグで誤検知が出ていた。v0.3.8 にアップデートで解消 |

## データ保存場所

- ログ: アプリ内 📋 モーダルのみ (ファイル保存なし)
