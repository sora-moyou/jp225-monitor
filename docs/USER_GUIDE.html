● Windows 10 / 11 · 無料 · v0.3.17

急変を 30 秒で
AI が日本語で説明する。

JP225 (日経 225) の価格変動をリアルタイムで監視。 v0.3.17 以降は 適応 z-score で急変動を検知 — 過去 60 分のボラティリティを基準に「いまの相場として異常か」を判定するので、固定 % 閾値より体感に近い感度になります。 日経 225 (NIY=F) には追加で 5〜10 秒の超短期フラッシュ検知 を装備、ニュース反応など瞬時の急変もキャッチ。

3 ステップ

3 分でセットアップ

ダウンロードから初回起動まで。API キーは無料で取得できます。

1

ダウンロード

GitHub Releases ページ を開きます。

📥 Releases ページの見方

ページを開いただけでは .exe は見えません。下にスクロール して Assets セクションを探してください。

  1. リリース説明の下、▶ Assets という折り畳みを クリックして展開
  2. 4 つのファイルが表示されるので、その中の 📦 JP225.Monitor_X.X.X_x64-setup.exe をクリック

※ .sig や latest.json はアプリ側が自動更新で使うもので、利用者は触る必要はありません。

2

インストール

ダウンロードした .exe をダブルクリック。

⚠ ここで多くの人が止まります

「Windows によって PC が保護されました」 という青い画面が出ます。 「実行しない」ボタンしか見えませんが、慌てないでください。

  1. 左上の小さな 詳細情報 リンクをクリック
  2. すると下に 実行 ボタンが出現するので、それをクリック

※ 無料配布アプリ全般で出る警告で、害はありません。気になる場合は GitHub のソースコード を確認できます。

ウィザードに従ってインストール → スタートメニューから JP225 Monitor 起動。

3

API キーを設定

起動時に設定モーダルが自動で開きます (未設定時)。後から開く時は 右上の ⚙️。

🔑 ここを忘れると AI 機能が動きません

AI 説明 / AI チャット / ニュース翻訳のために、API キーを 1 つ以上設定する必要があります。Gemini が無料で品質も十分。

  1. aistudio.google.com/apikey でキー発行 → コピー
  2. Gemini 欄に 貼り付け (空欄保存は無変更なので注意)
  3. 右下の 保存 をクリック
  4. 状態が 緑になれば完了

※ 詳しい操作は下の「設定画面の使い方」に画面付きで解説。

API キー

1 つだけ取れば動きます

3 つのプロバイダから 1 つ以上を選択。複数登録するとレート制限時に自動で次へフォールバック。

Gemini 2.5 Flash Lite — 1500 req / 日

  1. aistudio.google.com/apikey を開く
  2. Google アカウントでログイン
  3. 「Create API key」 → 表示された AIza... をコピー
  4. JP225 Monitor の設定モーダル「Gemini」欄に貼り付け → 保存
  5. 状態表示が 緑になれば完了

Groq Llama 3.3 70B — 14400 req / 日 (実質無制限)

  1. console.groq.com/keys を開く
  2. アカウント作成 → 「Create API Key」
  3. gsk_... をコピー → 設定画面に貼り付け

激しく使う場合のフォールバック用。Gemini と併設すると 429 時に自動切替。

OpenAI GPT-4o-mini — 従量課金 ($5〜)

  1. platform.openai.com/api-keys を開く
  2. クレジット入金 → 「Create new secret key」
  3. sk-... をコピー → 設定画面に貼り付け

Gemini と Groq の両方がレート制限に達した時のみ呼ばれます。最後の砦。

未設定でも動きます。 価格監視 / アラート音 / チャートは API キー無しでも動作。 無効になるのは「なぜ動いた」AI 説明、AI チャット、ニュース翻訳の 3 機能のみ。

設定画面の使い方

⚙️ アイコンを押してから API キー保存まで、6 ステップ。

右上の ⚙️ をクリック

トップバー右端のギアアイコン。初回起動時は API キー未設定なので自動で開きます。

状態を確認

モーダル上部に gemini / groq / openai が並びます。=設定済み / =未設定。

「キー取得 ↗」リンクをクリック

該当プロバイダのキー発行ページが別タブで開きます。Gemini 推奨。

キーをコピー

発行ページで AIza... / gsk_... / sk-... をコピー。

JP225 Monitor に戻り、対応欄に貼り付け

「設定済み (上書きする場合のみ入力)」プレースホルダの入力欄に貼り付け。空欄保存は無変更 なので、既存キーを残したい時は触らない。

右下の 保存 をクリック

状態表示が 緑に切り替わる。「閉じる」でモーダル退場。これで AI 説明・チャット・翻訳が即有効化。

保存先: 入力したキーは ~/.jp225-monitor/config.json にローカル保存されます。アプリ外部に送信されることはありません。
polling 設定: 下部の「定期 API ポーリング設定」で価格・ニュース取得間隔と port も変更可能。範囲外 (例 Price polling=100ms) は 400 で reject。
画面ツアー

どこに何があるか

起動直後の画面を 7 つの要素で説明します。

1 2 3 4 5 6 7 8
1
相関リーダー

NK225 と最も相関する銘柄を表示。5 分ごとに再評価され、その時々で最も連動性の高い指標がわかる。

2
受信中バッジ + API ドット

SSE 接続状態 + Y/G/Gr/O 各データソースの稼働状況。hover でカウントダウン表示。

3
📋 ログ・⚙️ 設定

サーバログ閲覧と全設定画面の入り口。API キー追加もここから。

4
価格カード

8 主要銘柄をリアルタイム更新 (2 秒間隔)。緑=上昇 / 赤=下落、急変時はハイライト点滅。「日経225先物」は NIY=F (CME 上場の JPY 建て日経 225 先物、価格は ~66,000 円台)。v0.3.8 以前は NK=F を使っていたが、これは Yahoo 側で 2020 年に更新停止した古い ALTSYMBOL だったため切替。

5
TradingView チャート

銘柄は FOREXCOM:JP225 (FOREX.com の Japan 225 CFD、JPY 建て)。価格カードの「日経225先物」(NIY=F) と 同じ日経 225 指数を別商品で表示しているだけ なので、絶対値も % 変動もほぼ一致します (差は売買スプレッド・取引時間帯のみ)。チャート上から銘柄・時間軸・指標を切替可能。

6
アラート + AI 説明

急変動を検知すると 30 秒以内に AI が要因を 1〜2 文で。🔄 で再生成、× で消去。

7
ニュース (21 ソース)

日本 7 + 英語 14 フィードを 60 秒間隔で集約。英語見出しは [訳] ボタンで日本語化可能。

8
AI チャット入力

「今の地合いは?」など自由質問。Enter で送信、Shift+Enter で改行。現在の全銘柄価格 + 直近 15 件ニュースが LLM コンテキストに自動付与されるので、相場固有の質問もそのまま聞ける。

💡 価格カード「日経225先物」とチャート銘柄の関係
どちらも 同じ日経 225 指数 を見ています。価格カード NIY=F (CME JPY 建て日経 225 先物) とチャート FOREXCOM:JP225 (FOREX.com の JPY 建て Japan 225 CFD) は商品種別こそ違いますが、参照指数は同一・通貨も同じ JPY なので、表示数字も % 変動もほぼ一致します。
機能一覧

できること

適応 z-score 急変検知 (v0.3.17)

1 分窓 burst (静寂前提付き) + 5 分窓 trend + 日経 225 専用 5〜10 秒 tick の 3 層構成。すべて過去 60 分のボラティリティ σ を基準にした 適応閾値 で発火するので、相場局面に追従。指数・FX は他銘柄との横断確認パスが必要 (単独データ異常を弾く)。

ニュース選別の質 (v0.3.9)

金融関連性判定をタイトル+本文ハイブリッドに、LLM へ渡すニュースは急変時刻 ±15 分 → ±60 分 → 4 時間の段階フォールバックで近接優先。的外れな材料引用を抑制。

trader-focused

AI 自動説明

4 時間分のニュースから関連性をスコアリングし、方向整合性 (リスクオン/オフ) を判定して因果を提示。

AI チャット

「今の地合いは?」など自由質問。現在価格と直近 15 件ニュースが自動でコンテキストに。

ニュース翻訳

on-demand

音声アラート

急変動でビープ音。上昇=高音 (1046 Hz)、下落=低音 (659 Hz)。初回のみ手動有効化が必要。

自動アップデート

起動 5 秒後に新版チェック。利用可能なら右上トースト通知 → ワンクリックで DL + 再起動。

値がさ株の裏監視

ファーストリテイリング / 東京エレクトロン / アドバンテスト / SBG / ダイキンの 5 銘柄も裏で監視 → 急変時にアラート。

カスタマイズ可

polling 間隔 (price 2s / news 60s) と sidecar port を設定モーダルから可変。範囲外は 400 で reject。

FAQ

よくある質問

アラート音が鳴りません
ブラウザのセキュリティ仕様で、初回起動時はユーザー操作が必要です。画面右下の 「🔔 サウンドを有効化」 ボタンを 1 回クリックすれば、以降は自動でビープが鳴ります。
「切断」のままで価格が出ない
  1. アプリを完全終了 (タスクトレイにアイコンがあれば右クリック → 終了)
  2. 再起動 (スタートメニューから)
  3. それでもダメなら、最新インストーラで上書きインストール (設定は保持されます)
アラートは出るのに「なぜ動いた」説明が表示されません
API ドットを確認:
  • 全部 = キー未設定 → ⚙️ から 1 つ以上設定
  • 全部 = 全プロバイダ 429 中 → 数分待つ。または別プロバイダ追加
Y (Yahoo) が黄色のまま
Yahoo Finance に拒否されている状態 (短時間に多数アクセスで 429)。価格データは Investing.com フォールバックが代替しているので 画面の価格は引き続き更新 されます。90s → 3min → 5min の段階で自動的に Yahoo を再試行。

v0.3.7〜v0.3.9 では 1 tick あたり Yahoo を 2 回呼んでいた残骸コードのせいで黄色から戻らない不具合が出ていました。v0.3.10 で単一バルク呼び出しに統合済み — 黄色固定はもう起きません。

価格カードが --- のまま
Yahoo と Investing.com の両方が同時失敗中。ネット接続を確認、数分待つ、ダメなら再起動。
起動時にファイアウォール許可ダイアログ
「アクセスを許可する」で OK。内部 HTTP サーバ (port 3000) のために必要です。プライベートネットワークのみで十分。
SmartScreen 警告は危険ではないですか?
無料配布アプリ全般で出る警告です。ソースコードは GitHub で全公開されているので、不安な場合は読んでから判断してください。「詳細情報」→「実行」で起動できます。
サーバログを見たい (デバッグ用)
トップバーの 📋 ボタンでログモーダルが開きます。直近 200 行を表示、新しいログが自動で下に追加。アプリ再起動で消えます。
polling 間隔を変えたい
⚙️ → 「定期 API ポーリング設定」セクション。
  • Price polling: 500–60000 ms (デフォルト 2000)。Yahoo 429 が頻発するなら 5000 推奨
  • News polling: 10000–600000 ms (デフォルト 60000)
  • Sidecar port: 1024–65535 (デフォルト 3000)。変更時はアプリ再起動必要
新版が出たらどうやって知る?
起動 5 秒後に updater が自動チェック。新版があれば右上にトースト通知が出ます。クリック 1 つで DL + 自動再起動 (約 30 秒)。「後で」は 24 時間後再通知。
更新時に「Error opening file for writing」エラーが出る
起動中のサイドカープロセスがファイルロックしているのが原因です。v0.3.3 以降のインストーラ で preinstall フックを追加し、書き込み前に自動 taskkill するので、もう出ません。

v0.3.2 以前を使っていてエラーに当たった場合:

  1. ダイアログで 「中止(A)」 をクリック (※「無視(I)」は絶対押さない — 旧 sidecar.exe が残り起動不能になる)
  2. Ctrl + Shift + Esc でタスクマネージャ → 「詳細」タブ
  3. jp225-sidecar.exe と jp225-monitor.exe を右クリック → タスクの終了
  4. 数秒待ってインストーラを再実行
日経 225 先物の表示価格が想定と違う
v0.3.10 以降は NIY=F (CME JPY 建て日経 225 先物) を使用しており、日経指数そのままの ~66,000 円台が表示されます。

v0.3.7 以前は NK=F を JPY 換算した ~1 千万円台が出ていましたが、これは Yahoo の NK=F が 2020 年に更新停止した古い ALTSYMBOL であり、JPY 換算自体も数学的に不要だったことが判明したため、v0.3.8 で銘柄切替・換算機構撤去を実施しました。

カードの価格はチャートと数字が違うんですが
v0.3.10 以降は ほぼ一致 します。価格カードの「日経225先物」(NIY=F) もチャート (FOREXCOM:JP225) も同じ日経 225 指数を JPY で表示しているため、差は売買スプレッド・取引時間帯の鮮度程度です。

v0.3.7 以前のバージョンを使っている場合は古い NK=F × USD/JPY 換算値 (~1 千万円台) になっているので、最新版にアップデートしてください。
