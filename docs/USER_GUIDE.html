JP225 Monitor 利用ガイド
導入 画面ツアー 売買シグナル アラート 設定 GitHub ↗
● Windows 10 / 11 · 無料 · v0.8.7

AI が値動きやニュースを監視し、
売買シグナルを提案する。

日経 225 先物 (OSE mini) をリアルタイムで監視。AI が相場状況を読んで買い/売りのエントリー(指値 + 逆指値 + 損切り)を提案し、 段階的に利益をロックする決済ロジックでシグナルの損益を自動で記録します。急変が起きたときは、その原因をニュースや他銘柄からたどって日本語で説明。 価格は 24/7 デーモンで蓄積され、起動時に即ウォームアップ。

3 ステップ

導入はかんたん

ダウンロード → インストール → API キーを 1 つ設定。以後はアプリ内「更新をチェック」で自動更新。Windows 10 / 11・無料です。

1

ダウンロード

配布は GitHub 上で行っています。まず GitHub アカウントを作成し、ログインしてからダウンロードしてください。 アカウントが無い方は github.com/signup で無料で作れます (メールアドレスとパスワードのみ)。ログイン後、GitHub Releases ページを開きます。

📥 Releases ページの見方

ページを開いただけでは .exe は見えません。下にスクロールして Assets を探してください。

  1. ▶ Assets の折り畳みをクリックして展開
  2. 署名済みの 📦 JP225.Monitor_X.X.X_x64-setup.exe をクリック

※ .sig や latest.json はアプリが自動更新で使うもので、利用者は触る必要はありません。

2

インストール

ダウンロードした .exe をダブルクリック。

⚠ ここで多くの人が止まります

「Windows によって PC が保護されました」という青い画面が出ます。「実行しない」しか見えませんが、慌てないでください。

  1. 左上の小さな 詳細情報 リンクをクリック
  2. 下に現れた 実行 ボタンをクリック

※ 無料配布アプリ全般で出る警告で、害はありません。ウィザードに従って進めれば完了です。

3

API キーを 1 つ設定

初回起動時に設定モーダルが自動で開きます。後から開くときは右上の ⚙️。

🔑 これを忘れると AI 機能が動きません

AI 急変説明 / AI チャット / ニュース翻訳のために、API キーを 1 つ以上設定します。Gemini が無料でおすすめ。

  1. aistudio.google.com/apikey でキー発行 → コピー
  2. Gemini 欄に貼り付け(空欄保存は無変更なので注意)
  3. 右下の 保存 をクリック
  4. 状態が 緑になれば完了

※ 詳しくは下の「設定 — API キー」を参照。

以後の更新は自動。 インストール後は起動時に新版を自動チェックし、⚙️ 設定の「更新をチェック」からいつでも手動確認もできます。
画面ツアー

どこに何があるか

起動直後の画面。売買シグナルを中心に、価格カード・主要レベル・AI チャット・アラート・チャート・ニュースが 1 画面に収まります。

1
価格カード

日経 225 先物(NIY=F・OSE リアルタイム)、ナスダック 100 先物などを更新。日経の値の上に「1 分」(率 %)と「10 秒」(値幅・円)の 2 段モメンタムを表示します。

2
売買シグナル

現在のシグナル(買い/売り・指値/逆指値+LC)を大きな文字で表示。提案の理由は白文字で示します。

3
主要レベル(節目)

★ 点数つきの上値/下値メド。セッション高安・フィボナッチ・反応水準・下降/上昇トレンドライン(合流)・出来高帯・もみ合い帯・日足 MA5/20/50/75・日足バンド(MA25±1σ/±2σ)を、現値位置でソートして表示。

4
AI チャット

①テクニカル分析 ②ファンダメンタル分析 ③スイング分析 のプリセット + 自由入力。市場の現状(価格/節目/アラート/本日 OHLC/ニュース)を踏まえて回答。キー設定時は Web 検索も可能です。

5
ALERTS(アラート + AI 説明)

急変・暴落・水準抜け/反発・ダブルトップ/ボトム・トレンド転換・グランビル・日足 MA/バンドなどの異常を検知し、AI が原因を日本語で説明。事後の値動き(+5/15/30 分)も記録します。

6
チャート

TradingView(Japan 225 CFD)を埋め込み表示。時間足を切り替えられます。

7
NEWS

日英のニュースを集約表示。英語見出しは [訳] ボタンで日本語化できます。

8
🔔 サウンドを有効化

右下のボタン。ブラウザの制約で、初回に 1 回クリックすると通知音が有効になります。

主役の機能

AI が売買シグナルを提案

AI が値動きやニュースを監視して、買い/売りのエントリー(指値 + 逆指値 + 損切り)を提案。段階的に利益をロックする決済で、シグナルの損益を自動で記録します。

▲ 表示例 — 向き・エントリー価格・指値/逆指値・損切り(LC)と、AI がまとめた提案理由が画面左上に表示されます。

① AI がエントリーを提案

相場状況を読んで、買い/売りの向きと指値 + 逆指値 + 損切り(LC)のプランを提示。現在のシグナルは画面左上のパネルに大きく表示され、発生時にはアラート音でも知らせます。

② phase-exit で段階的に利益確定

含み益に応じて利益を段階的にロックする決済ロジックで、シグナルの損益を集計します。

③ 履歴・収益曲線を確認

📈 ボタンで履歴と収益曲線(件数・勝率・累積損益 + 一覧)を表示。再起動しても履歴は残り、設定でシグナル発生時のアラート音 ON/OFF・履歴消去ができます。

補助 — アラート

急変を検知し、AI が原因を説明

シグナルの判断材料として、日経 225 先物の異常を検知。値動き系は AI が原因を日本語で説明、テクニカル系は根拠付きの固定文で表示します。

急変・暴落(値動き系)

急変 — 完成 1 分足の価格変化スコア(絶対値幅 + 直近ボラ比)。
暴落(crash) — セッション高値から一定 % の下落。
いずれも AI がニュース・他資産から原因を「ファンダ要因かテクニカル要因か」判定して説明します。

水準系(テクニカル)

水準抜け・反発(意識される節目のブレイク/反発)、ブレイク、ダブルトップ・ボトム、トレンド転換(MA 抜け + 傾き反転)。すべて確定スイングと ★ 水準のみを対象に、価格/MA を併記して表示します。

グランビル・日足系

グランビル(押し目買い・戻り売り・転換)、日足バンド(MA25 ±σ)、日足 MA5/20/50/75 のサポート/レジスタンス。中長期の節目に対する反応を捉えます。

事後の値動きも記録: 各アラートの発生後の値動き(+5/15/30 分)を自動で記録し、AI の原因説明とあわせて振り返りに使えます。
設定 — API キー

キーは 1 つで動きます

⚙️「設定」から。無料キーだけで十分に使えます。複数登録するとレート制限時に自動でフォールバック。

無料キーから設定

Gemini(画像対応・日次 1500 回・おすすめ)/ Groq(テキスト専用・高速・日次 14400 回)。どちらも無料。

安定運用には OpenAI(任意・有料)

画像対応で、Gemini が枠切れ(429)のときの最後の砦。少額の課金で長く使えます。もう 1 つの Gemini キーはチャットの Web 検索用に使えます。

「キーを検証」で確認

各キー行の状態: 🟢 有効 / 🟡 待機中(429) / ⚪ 未設定。検証ボタンで実際に通るか確認できます(✅ 有効 / ❌ 無効)。

Gemini — 画像対応・日次 1500 回・おすすめ

  1. aistudio.google.com/apikey を開く
  2. Google アカウントでログイン → 「Create API key」
  3. 表示された AIza... をコピー → 設定の「Gemini」欄に貼り付け → 保存

Groq — テキスト専用・高速・日次 14400 回

  1. console.groq.com/keys を開く
  2. アカウント作成 → 「Create API Key」
  3. gsk_... をコピー → 設定画面に貼り付け

テキスト専用のため、チャート画像を見る AI(トレードシグナル)では使われません。チャット/説明の予備として無料で有用。

OpenAI — 画像対応・従量課金(少額)

  1. platform.openai.com/api-keys を開く
  2. 少額のクレジット入金 → 「Create new secret key」
  3. sk-... をコピー → 設定画面に貼り付け

画像対応 + 最後の砦。有効にしておくと画像判断が常に通り、運用が安定します。

保存先はローカルのみ: 入力したキーは ~/.jp225-monitor/config.json に保存され、アプリ外部に送信されることはありません。
未設定でも動きます。 価格監視・アラート音・チャートはキー無しで動作。無効になるのは AI 説明・AI チャット・ニュース翻訳の 3 機能だけです。
設定 — 更新 / 終了 / シグナル音

更新・終了時の挙動

アプリ本体と基礎データの更新、終了時のデーモン停止、トレードシグナル音を設定します。

更新をチェック

アプリ本体 + 基礎データ(1 分足履歴)の新着を確認・取り込み。既存データは消えず、追記/更新のみで安全です。

完全終了

チェックを ON にすると、終了時にバックグラウンドの価格収集デーモンも停止します。既定(OFF)なら、閉じても収集を継続し、次回起動時に即ウォームアップできます。

売買シグナル

シグナル発生時のアラート音 ON/OFFと履歴の消去ができます。

リファレンス

取引時間・基礎データ

取引時間(OSE・2024 年 11 月延長後)
  • 日中(Day): 08:45 – 15:45
  • 夜間(Night): 17:00 – 翌 06:00
  • 取引日足: 夜間 17:00 〜 翌日中 15:45(終値 = 15:45)
価格データ(24/7 蓄積)
価格は 24/7 の収集デーモンが SQLite に蓄積し、アプリ起動時に即ウォームアップします。これにより主要レベルや急変判定が起動直後から揃います。取引時間外は各更新ループを自動休止し、セッション開始前に自動再開します。
